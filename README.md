# ai-chat-keymap

ChatGPT, Claude, GeminiのプロンプトInputでキーボードのキーマップを変更するChrome拡張機能。

## キーマップ

| キー | 動作 |
|------|------|
| `Enter` | 改行 |
| `Shift + Enter` | 改行 |
| `Cmd + Enter` (Mac) | 送信 |
| `Ctrl + Enter` (Windows/Linux) | 送信 |

## 対応サイト

- [ChatGPT](https://chatgpt.com) / [chat.openai.com](https://chat.openai.com)
- [Claude](https://claude.ai)
- [Gemini](https://gemini.google.com)

## インストール方法

1. このリポジトリをクローン、またはZIPでダウンロードして展開します。
2. Chrome で `chrome://extensions` を開きます。
3. 右上の **デベロッパーモード** を有効にします。
4. **パッケージ化されていない拡張機能を読み込む** をクリックし、このフォルダを選択します。
5. 対応サイトを開くとキーマップが自動的に適用されます。
